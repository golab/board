About

This is an online board which allows shared control across all participants. Create a board, share the link with your friends. Experiment with variations. In theory, you could even play informal games.

How to Use

Enter a name for your new board (or leave it blank and the server will generate an id for you), and click "New Board." This will generate a new URL that you can then share with other participants.
After 24 hours of no activity, boards will be automatically cleared.

Arrow Keys

The left and right arrow keys move left and right in the game tree. The up and down arrow keys move up and down branches (a la OGS), while shift+up and shift+down modify which branch to follow (a la KGS). This functionality can be switched in the settings by clicking on the button.

Input Buffer

With no other controls in place, users will quickly start "interrupting" each other when putting stones on the board at the same time. So, each room has a slider for what we call "Input Buffer". After a user changes the board in some way, the server will ignore everyone else for a given period of time (default: 250ms). This can be adjusted in the settings.
