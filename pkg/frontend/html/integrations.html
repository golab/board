Integrations

Twitch

If you are a Twitch streamer, you can connect a board to your Twitch chat.

Authorize and subscribe the chatbot to your channel. You must be logged in to Twitch.

Unsubscribe the chatbot from your channel. You must be logged in to Twitch.

As the channel owner, use the command

!setboard [board_id]

in chat to specify which board to connect to. Audience members can use the command

!branch [coordinates]

to add a branch to the tree. For example, use

!branch e17 d17 e16 c14

to add a branch starting at the current node. Audience members can also use 

!branch [node] [coordinates]

to add branches starting at a given node. For example, use

!branch 15 e17 d17 e16 c14

to add a branch starting at node 15.
