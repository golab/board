## Features
- Shared control of the go board
- User-specified URLs
- No sign-up or login
- Password-protected rooms
- Browser-based
- Desktop and mobile compatible
- File formats accepted: SGF, GIB, NGF, ZIP
- Sync with live OGS games or reviews
- Integrate with Twitch
- Free and open source
- Open license [asset pack](https://github.com/golab/board/raw/refs/heads/test/assets/assets.zip) (board and white stones)
