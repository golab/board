A free and open-source multi-user go board.

![dark mode screenshot](docs/img/darkmode.png)

## Features
- Shared control of the go board
- User-specified URLs
- No sign-up or login
- Password-protected rooms
- Browser-based
- Desktop and mobile compatible
- Sync with live OGS games
- Integrate with Twitch
- Free and open source
- Open license [asset pack](https://github.com/jarednogo/board/raw/refs/heads/test/assets/assets.zip) (board and white stones)
